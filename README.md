# PyArkan
Arkanoid like game, written by Python

![Screenshot](https://raw.githubusercontent.com/An0ther0ne/PyArkan/master/scrshot.png)

To start press Space Bar.
Other controll keys are simple: 
Left or A  – move left
Right or D – move right

## Requirements

* Python
* PyGame
